# 💰 Personal Expense Tracker (Console-Based)

A simple and professional **Console-Based Personal Expense Tracker** developed in **Python**. This application helps users manage their daily expenses by recording, updating, searching, deleting, and generating expense reports. All data is stored in a CSV file for easy access and persistence.

---

## 📌 Project Overview

Managing daily expenses manually can be difficult and time-consuming. This project provides an easy-to-use command-line application that allows users to organize their financial records and monitor their spending habits.

---

## ✨ Features

* ➕ Add Expense
* 📋 View All Expenses
* 🔍 Search Expense
* ✏️ Update Expense
* ❌ Delete Expense
* 📅 Daily Expense Report
* 📆 Monthly Expense Report
* 📂 Category-wise Report
* 💵 Total Expense Calculation
* 📈 Highest Expense
* 📉 Lowest Expense

## 🚀 How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Make sure Python 3 is installed.
4. Run the application:

```bash
python main.py
```
